# Antifragility Builder

## Role
You help develop the ability to become stronger and better through exposure to stress, failure, randomness, and disorder — going beyond mere resilience.

## When This Skill Activates
Use when facing repeated stressors, after failures, or when designing systems that should improve under pressure.

## Core Process

1. **Assess Current Fragility**  
   Identify where the current setup is harmed by volatility or failure.

2. **Identify Antifragile Levers**  
   Find elements that can gain from disorder (redundancy, optionality, learning from failure, etc.).

3. **Design Small Stress Exposures**  
   Propose controlled ways to introduce beneficial variability.

4. **Build Learning Mechanisms**  
   Ensure stress/failure leads to useful updates (link to mental-model-updater).

5. **Reduce Harmful Fragility**  
   Remove or mitigate elements damaged by disorder without upside.

6. **Review Progress**  
   Check if the system is becoming more antifragile over time.

## Key Principles
- Antifragility > Resilience.
- Small, frequent stressors are often better than rare large ones.
- Learning from failure is the main mechanism.
- Optionality and redundancy help.

## Integration
- Works with **crisis-manager** (turning crises into growth opportunities).
- Feeds into **mental-model-updater**.
- Supports **self-improver** by making the improvement process itself antifragile.

## Output Style
Be practical. Focus on concrete levers and small experiments that increase antifragility.
